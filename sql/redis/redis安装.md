#### win10下安装redis
1. 安装redis
[redis](https://github.com/MicrosoftArchive/redis/releases)

2. 安装RedisPy
    ```
    pip install redis
    ```
3. 安装RedisDump
    1. 安装ruby
        [ruby](http://www.ruby-lang.org/zh_cn/documentation/installation/)
    2.  ```
        gem install redis-dump
        ```
